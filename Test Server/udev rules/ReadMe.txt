The 99-coreboot-ADTS.rules file contained in this folder is a udev rules file that helps to get 
persistent symbolic links to the devices involved in coreboot automatic distributed test system (ADTS). 

The involved devices are test interface boards, power strips and flash programming devices

.

Please edit this template to match the configuration of your set-up and the identification parameters 
of your devices and save it to the folder /etc/udev/rules.d/
